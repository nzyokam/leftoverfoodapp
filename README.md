# 🌍 Leftover Food Share App - PROJECT BY MUUSI NGUUTU NZYOKA

A Flutter-powered mobile application designed to combat hunger and food waste by connecting surplus food donors with nearby shelters, volunteers, and community aid hubs.

## 📱 Project Overview

**Status**: Planning Phase (Construction Has Begun)  
**Type**: Individual Project  
**Platform**: Flutter (primarily developed for iOS)  
**Purpose**: Social Impact Application

## 🎯 Mission Statement

To fight hunger by creating a bridge between surplus food and those who need it most, while reducing food waste and building stronger communities through technology and compassion.

## 👥 Target User Roles

- **Food Donors**: Restaurants, cafés, grocery stores with surplus food
- **Recipient Hubs**: Shelters, soup kitchens, community centers
- **Volunteer Runners**: Individuals who deliver food to those in need
- **Platform Admins**: Oversight, moderation, and analytics management

## ✨ Planned Features

### 🍲 Core Functionalities

#### Food Donation & Listing
- Quick food posting with image capture
- Detailed descriptions with allergen tags
- Expiry and pickup time windows
- Auto-location detection
- Real-time listing management

#### Community Pickup System
- Map and list view of available food
- Advanced filtering (type, quantity, distance, expiry)
- Claiming and pickup scheduling
- Integrated navigation and contact options
- Pickup confirmation and rating system

#### Digital Inclusion Features
- Shelter/NGO representative access for homeless users
- SMS alerts for basic phone users
- Volunteer delivery coordination
- Community kiosk integration

### 📊 Impact & Engagement

#### Impact Dashboard
- Meals saved counter
- CO₂ offset calculations
- Pickup completion tracking
- Volunteer badge system
- Shareable impact statistics

#### Gamification Elements
- Donor verification badges
- Volunteer level progression
- Monthly regional leaderboards
- Community hero recognition

### 🛡️ Safety & Accessibility

#### Food Safety Guidelines
- Pre-pickup safety checklist
- Food handling education for volunteers
- Expiry date verification
- Proper packaging requirements

#### Accessibility Features
- Multi-language support (English, Swahili, Spanish)
- High-contrast, low-literacy friendly UI
- Offline-capable design
- Simple navigation patterns

## 🔔 Notification System

- Push notifications for food availability
- SMS alerts for basic phone users
- Pickup reminders
- Urgent expiry notifications

## 🗺️ User Journey Examples

### Food Donor Flow
1. Register as donor → Verify business
2. Post surplus food with details
3. Receive pickup requests
4. Confirm pickup completion
5. View impact statistics

### Shelter/NGO Flow
1. Register as recipient hub
2. Browse available food nearby
3. Claim needed items
4. Schedule pickup time
5. Distribute to community

### Volunteer Runner Flow
1. Register as volunteer
2. View delivery requests
3. Accept pickup/delivery task
4. Navigate to locations
5. Confirm delivery completion
